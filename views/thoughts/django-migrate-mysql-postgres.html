Migrate a MySQL Django Database to PostgreSQL

Dump data from your current MySQL database as JSON: 

      python manage.py dumpdata mysql_data.json

Create a new PostgreSQL database if you haven't already, update your project settings.py file and sync the database with Django to create the database tables: 

      python manage.py syncdb

Wipe out all data in the new database: 

      python manage.py sqlflush | psql -U username db

Import the JSON we dumped from MySQL into PostgreSQL: 

      python manage.py loaddata mysql_data.json

That's it, have fun!
